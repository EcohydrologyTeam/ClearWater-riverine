Welcome to Clearwater Riverine's documentation!
===============================================

**Clearwater Riverine** is a Python 3 library that enables users to simulate the transport 
(advection and diffusion) of constituents in riverine systems. 
Clearwater Riverine leverages HEC-RAS 2D output, which provides the model grid and hydrodynamic
information to the Clearwater Riverine system. 

.. note::

   This project is under active development.

Use and Citation
=================
ClearWater is developed by the Environmental Laboratory, 
U.S Army Engineer Research and Development Center (ERDC)
in collaboration with private sector contractors.
